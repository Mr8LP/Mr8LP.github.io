ANSYS-BASIC

Author: Yan Hao Date: January 25, 2023  10:10:03 Category: ANSYS

利用ANSYS软件进行有限元分析的步骤

指定文件名和标题名——定义单元属性——定义材料属性——创建有限元模型——对实体模型进行网格划分——施加约束和载荷——求解

ANSYS软件分析的两种方法

  1. GUI 方法：直接操作软件界面，简单直观，交互性强，但是效率偏低
  2. 命令流方法：采用APDL语言创建命令流，难度较大，交互性差，效率较高，常利用 Visio Code，Ultra Edit，notepad++进行编辑，详情查看ANSYS help文档查看

前处理命令流

finish

GUI : main menu-finishExits

Description : any of the ANSYS processors or the DISPLAY program. For the ANSYS processors, data will remain intact in the database but the database is not automatically written to a file (use the SAVE command to write the database to a file)

/clean

Description : It can be used by the user to perform clean-up operation or other actions that are needed at the end of a job.

/prep7

GUI ：Main Menu>Preprocessor

Description ： Enters the model creation preprocessor.

！

Description ： annotation symbol，used for code annotation or further explanation .

et(定义单元类型)

GUI : Menu Paths:Main Menu>Preprocessor Element Type>Add/Edit/Delete

Description : ET,ITYPE,Ename,KOPT1,KOPT2,KOPT3,KOPT4,KOPT5,KOPT6,INOPR

ET ITYPE Ename (ommited) KOP1-7 (ommited) INOPR (ommited)
Command name Arbitrary local element type number. Element name (or number) given in the element library,The name consists of a category prefix and a unique number, such as PIPE288. KEYOPT values (1 through 6) for this element If 1, suppress all element solution printout for this element type.

mp(定义材料属性)

GUI : Main Menu>Preprocessor>Material Props

Description : MP, Lab, MAT, C0, C1, C2, C3, C4

MP Lab MAT C0 C1-C4
Command name Valid material property label. Such as: dens, prxy,ex… Material reference number to be associated with the elements Material property value Coefficients of the linear, quadratic, cubic, and quartic terms, respectively

sectype(定义材料截面属性)

GUI : Main Menu>Preprocessor>Sections

Description : SECTYPE, SECID, Type, Subtype, Name, REFINEKEY

SECTYPE SECID Type Subtype Name REFINEKEY
Command name Section identification number. Beam, Link, Taper… subtype An eight-character name for the section. Sets mesh refinement level for thin-walled beam sections. Valid values are 0 (the default - no mesh refinement) through 5 (high level of mesh refinement). This value has meaning only when Type = BEAM

secdata(定义材料截面尺寸)

GUI : Main Menu>Preprocessor>Sections

Description : SECDATA, VAL1, VAL2, VAL3, VAL4, VAL5, VAL6, VAL7, VAL8, VAL9, VAL10, VAL11, VAL12

Describe the geometry of a section. The terms VAL1, VAL2, etc. are specialized for each type of cross-section.

创建有限元模型命令流

k(定义关键点)

GUI : Main Menu>Preprocessor>Modeling>Create>Keypoints

Description : K, NPT, X, Y, Z

K NPT X,Y,Z
Command name Reference number for keypoint. If zero, the lowest available number is assigned,mini=1 Keypoint location in the active coordinate system

L(从点创建线)

GUI : Main Menu>Preprocessor>Modeling>Create>Lines>Lines>In Active Coord

Description : L, P1, P2, NDIV, SPACE, XV1, YV1, ZV1, XV2, YV2, ZV2

L P1 P2 NDIV-ZV2（ommited）
Command name Keypoint at the beginning of line Keypoint at the end of line. look up ANSYS help

对实体模型进行网格划分

latt(分配直线网格属性)

GUI : Main Menu>Preprocessor>Meshing>Mesh Attributes

Description : LATT, MAT, REAL, TYPE, –, KB, KE, SECNUM

LATT MAT REAL TYPE – KB, KE SECNUM
Command name Material numbe real constant set number type number Unused field Beginning and ending orientation keypoints to be associated with selected, unmeshed lines. section number

lesize(指定未网格化线的间距及间距比)

GUI : Main Menu>Preprocessor>Meshing>Size Cntrls>ManualSize

Description : LESIZE, NL1, SIZE, ANGSIZ, NDIV, SPACE, KFORC, LAYER1, LAYER2, KYNDIV

lmesh

GUI : Main Menu>Preprocessor>Meshing>Mesh>Line

Description : LMESH, NL1, NL2, NINC

Mesh lines from NL1 to NL2 in steps of NINC (defaults to 1). If NL1 = ALL, NL2 and NINC are ignored and all selected lines are meshed.

施加约束和载荷

dk(关键点上施加约束)

GUI : Main Menu>Preprocessor>Loads>Define Loads>Apply

Description : DK, KPOI, Lab, VALUE, VALUE2, KEXPND, Lab2, Lab3, Lab4, Lab5, Lab6

DK KPOI Lab VALUE VALUE2, KEXPND, Lab2-6
Command name Keypoint at which constraint is to be specified/all Valid degree of freedom label,Ux-z, ROTx-z,all Degree of freedom value ANSYS help

fk(关键点上施加载荷)

GUI : Main Menu>Preprocessor>Loads>Define Loads>Apply

Description : FK, KPOI, Lab, VALUE, VALUE2

FK KPOI Lab VALUE, VALUE2
Command name Keypoint at which force is to be specified/all Valid force label,Fx-z,Mx-z Force or moment value
