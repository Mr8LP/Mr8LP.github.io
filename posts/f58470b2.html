ANSYS-BASIC

Author: Yan Hao Date: January 25, 2023  10:10:03 Category: ANSYS

简介

利用ANSYS软件进行有限元分析的步骤

指定文件名和标题名——定义单元属性——定义材料属性——创建有限元模型——对实体模型进行网格划分——施加约束和载荷——求解

ANSYS软件分析的两种方法

  1. GUI 方法：直接操作软件界面，简单直观，交互性强，但是效率偏低
  2. 命令流方法：采用APDL语言创建命令流，难度较大，交互性差，效率较高，常利用 Visio Code，Ultra Edit，notepad++进行编辑，详情查看ANSYS help文档查看

坐标系

坐标系的分类

全局坐标系(Global CS)

简介：主要分为笛卡尔坐标系、柱坐标系和球柱坐标系，默认激活为笛卡尔坐标系

相关命令：用于激活坐标系-(csys, num)，num=0、1、2、5

工作平面坐标系(Working Plane CS)

简介：2-D作图平面，主要用于切割图元和基本图元（点线面）的定位和定向，仅能是笛卡尔坐标系，默认与全局坐标系的XOY平面重合

相关命令：用于激活坐标系-(csys, 4)

局部坐标系(Local CS)

简介：任意位置创建的坐标系，可以绕坐标轴旋转，类型可以是笛卡尔坐标系、柱坐标系和球柱坐标系

相关命令：用于激活坐标系-(csys, num)，num>=10，local

LOCAL(通过位置和方向定义局部坐标系)

GUI : Utility Menu>WorkPlane>Local Coordinate Systems>Create Local CS>At Specified Loc

Description : LOCAL, KCN, KCS, XC, YC, ZC, THXY, THYZ, THZX, PAR1, PAR2

节点坐标系

简介：方便特定荷载的施加，默认平行于当前激活的坐标系

相关命令：Nrotat

结果坐标系

简介：所有的后处理都在此坐标系下，默认下，软件会将所有结果旋转到全局直角坐标系

基本图元的修改

GUI : Main Menu>Preprocessor>Modeling

命令流 : Kdele, Ldele, Adele, Vdele, Kgen, Lgen, Vgen, Lsymm, Vsymm

Ldele(删除未网格化的线条)

GUI : Main Menu>Preprocessor>Meshing>Concatenate>Del Concats>Lines

Description : LDELE, NL1, NL2, NINC, KSWP

LDELE NL1, NL2, NINC KSWP
Command name Delete lines from NL1 to NL2 (defaults to NL1) in steps of NINC (defaults to 1)/ All Specifies whether keypoints are also to be deleted: if KSWP=0, Delete lines only, but if KSWP=1, Delete lines, as well as keypoints attached to lines but not attached to other lines.

Lgen(复制/移动、旋转已知线)

GUI : Main Menu>Preprocessor>Modeling>Copy>Lines

Description : LGEN, ITIME, NL1, NL2, NINC, DX, DY, DZ, KINC, NOELEM, IMOVE

LGEN ITIME (>=1) NL1, NL2, NINC DX, DY, DZ KINC NOELEM IMOVE
Command name Do this generation operation a total of ITIMEs, incrementing all keypoints in the given pattern automatically (or by KINC) each time after the first. Generate lines from pattern beginning with NL1 to NL2 (defaults to NL1) in steps of NINC (defaults to 1) /All Keypoint location increments in the active coordinate system Keypoint increment between generated sets. If zero, the lowest available keypoint numbers are assigned Specifies if elements and nodes are also to be generated (0/1) Specifies whether to redefine the existing lines (0/1), if IMOVE = 1, ITIME, KINC, and NOELM are ignored

命令具体是移动还是旋转要视激活的坐标系类型而定，如果激活的是笛卡尔坐标系，那就是移动操作；如果激活的是柱坐标系，那就是旋转操作。
